Tutorial de Canvas no HTML5

O elemento <canvas> é usado para desenhar gráficos via JavaScript. Neste tutorial, vamos mostrar como configurar e desenhar no canvas.

Passo 1: Adicionando o elemento Canvas

Primeiro, adicione o elemento <canvas> no seu HTML.

<canvas id="meuCanvas" width="500" height="400">Seu navegador não suporta o elemento canvas.</canvas>
Seu navegador não suporta o elemento canvas.

Passo 2: Acessando o contexto do Canvas

Depois de adicionar o canvas, você precisa acessar o contexto de desenho com JavaScript.

const canvas = document.getElementById('meuCanvas');
const contexto = canvas.getContext('2d');

Passo 3: Desenhando no Canvas

Agora, você pode usar o contexto para desenhar. Aqui está um exemplo de como desenhar um retângulo e um círculo.

// Desenhando um retângulo
contexto.fillStyle = '#FF0000';
contexto.fillRect(20, 20, 150, 100);

// Desenhando um círculo
contexto.beginPath();
contexto.arc(300, 200, 50, 0, 2 * Math.PI);
contexto.fillStyle = '#00FF00';
contexto.fill();
